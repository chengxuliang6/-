
/*正压工作部分
此直流稳压电源采用TPS7A4701&TPS7A3301
TPS7A4701输入电压+3到+36
可以使用拨码开关或外置反馈电阻来调整输出电压。
在第一版的PCB中通过拨码开关U10控制U1或U2工作与否；
在U1工作时可以通过八位拨码开关调节输出电压       //输入电压最好大于输出电压否则可能会造成输出电压不稳，PCB中的丝印是一个保守值，可以尽量的大5-7V
在U2工作时通过四位拨码开关U3控制U2 工作在FeedBack/sense模式    //sense模式即拨码开关控制电压
在使用FB模式是请将一位拨码SW1开关闭合，与此对应在SENSE模式将其断开	//FB模式即通过外置电阻调节电压公式请去看附图1
在第一版PCB中U2的最大输出电压大约是35V，但是不建议调节到35V我的建议是输入电压是输出电压的1.414倍

/*负压工作部分
TPS7A3301输入电压-3到-36V
可以使用外置反馈电阻来调整输出电压。//详情请看附图2
在第一版的PCB中通过拨码开关U11控制U6或U7工作与否
U6的输出电压在-1.8到-19V
U7的输出电压在-14到-32V
负压工作部分的输入电压建议与正压工作部分相同

两个47uf的铝电解电容必焊//C5与铝电解电容选焊

U6C22，10nf
C21，1uf
